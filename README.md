# julklappslekar.se
Julklappslekar.se är en sajt där reglerna till julklappsspelet presenteras, olika varianter på leken och tips på paket.

## Tech stack
Tech-stacken består av React, Next.js, Apollo, GraphQL, Prisma 2.
